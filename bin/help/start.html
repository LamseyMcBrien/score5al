About Score5al

Score5al is a scoreboard program for roller derby competitions using the Sur5al format. It was written for the Royal Windsor Rollergirls by Liam McBrien. For acknowledgements and copyright information, please see the program's "About" dialog, which is accessible from the Help menu.
